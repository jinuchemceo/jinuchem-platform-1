# PRD — JINUCHEM 통합 플랫폼

> **버전:** 2.0
> **작성일:** 2026-03-19
> **작성자:** JINUCHEM 개발팀
> **상태:** 프로덕션 v1.0 개발 완료 (5개 앱 + DB 42 모델 + AI 4엔진)

---

## 1. 제품 개요

### 1-1. 제품명
**JINUCHEM 통합 플랫폼** — B2B 연구기관 시약/소모품 커머스 + 연구 데이터 플랫폼

### 1-2. 비전
"연구자가 실험에만 집중할 수 있도록, 시약 구매부터 연구 데이터까지 하나의 생태계에서"

JINU Shop을 모객 창구(Customer Acquisition Gateway)로 삼아 연구기관 사용자를 확보하고,
거래/검색/가격 데이터를 수집하여 AI 기반 2차 데이터를 생성한 뒤 외부 API로 제공하는
**연구 데이터 플랫폼**으로 확장.

### 1-3. 타겟 사용자

| 역할 | 페르소나 | 핵심 니즈 |
|------|---------|---------|
| **연구원** | 대학원생·박사후·기업 연구원 | 빠른 검색, 간편 주문, 재주문 |
| **조직관리자** | 교수·연구실장·팀장 | 예산 관리, 구매 승인, 지출 분석 |
| **공급자** | 시약 유통사·제조사 | 주문 접수, 견적 응답, 배송 관리 |
| **시스템관리자** | JINUCHEM 운영팀 | 전체 플랫폼 운영, 공급사 관리 |

### 1-4. 핵심 가치 제안
1. **원스톱 구매** — 다수 공급사(Sigma-Aldrich, TCI, Alfa Aesar 등) 비교·통합 주문
2. **연구 특화** — CAS번호 검색, 구조식 검색, GHS 위험정보, SDS/COA 문서
3. **조직 워크플로우** — 역할별 권한, 구매 승인, 예산 관리, 증빙서류 자동 발급
4. **지누 E-Note 연동** — 전자노트와 시약장 양방향 동기화
5. **데이터 플랫폼** — 거래/검색/가격 데이터 수집 → 연구 구매 인사이트 제공
6. **AI 2차 데이터** — 시약 추천, 소모량 예측, 분석 리포트, 화학 지식 그래프

## 2. 기능 요구사항

### 2-1. 제품 검색 및 탐색

#### FR-001. 통합 검색
- 시약명, CAS번호, 분자식, 카탈로그번호, 키워드로 검색
- CAS번호 패턴(`XX-XX-X`) 자동 감지 → 정확 매칭 우선
- 자동완성 드롭다운 (제품명 + 카탈로그번호 + 카테고리)
- 검색 결과 개수 표시

#### FR-002. 구조식 검색
- Draw 모드: 화학 구조식 그리기 도구
- SMILES 입력: 텍스트 기반 구조식
- 분자식 입력: 원소 조합 검색
- 유사 구조 검색 (서브스트럭처 매칭)

#### FR-003. 필터링
- 배송: 당일출고 가능 필터
- 카테고리: 유기화합물, 무기화합물, 생화학시약, 분석용시약, 용매, 고분자 등
- 등급: ACS Grade, HPLC Grade, GR Grade, 특급, 1급
- 공급사: 다중 선택 + 공급사 목록 내 검색
- 가격 범위: 슬라이더 또는 직접 입력

#### FR-004. 정렬 및 뷰
- 정렬: 인기순, 가격낮은순, 가격높은순, 최신순
- 뷰 전환: 그리드(4열) / 리스트(1열)
- 페이지네이션: 페이지당 20개, 원형 번호 버튼

### 2-2. 제품 정보

#### FR-005. 시약 상세
- 구조식 이미지 (확대 가능)
- 기본 정보: 제품명, CAS No., 분자식, 분자량, 순도, 등급
- 동의어 표시
- 공급사명 + SKU
- GHS 위험 분류: 픽토그램, H-code, 위험문구
- 포장단위별 가격 테이블 (사이즈, 정가, 할인가, 재고, 납품예정일)
- 문서: SDS(한글/영문), COA, COO, Specification Sheet

#### FR-006. 소모품 상세
- 제품 사양 테이블
- 호환 장비 가이드
- 정기배송 옵션 (주기 설정: 1주/2주/1개월/2개월/3개월)
- 대체품 추천
- 문서: Spec Sheet, KC 인증서, SDS

#### FR-007. 가격 표시
- 정가/할인가 이중 표시 (정가 취소선 + 할인가 강조)
- 할인율 배지
- VAT 자동 계산
- 포장단위 선택 시 가격 즉시 변경

### 2-3. 주문 프로세스

#### FR-008. 장바구니
- 제품 추가/삭제/수량 변경
- 장바구니 담기 토스트 알림
- 프로모션 코드 입력
- 장바구니 이메일 공유 / PDF 다운로드 / 인쇄
- "카탈로그번호 직접 추가" 입력란 (Merck 방식)
- 소계 → 할인 → VAT → 총액 자동 계산

#### FR-009. 빠른 주문
- 검색 유형 선택: 제품번호, 제품명, CAS No., 분자식, 키워드
- 입력 → 자동완성 → 테이블에 추가
- CSV/엑셀 파일 대량 업로드
- 특이사항 표시 (항공규제, 드라이아이스 등)

#### FR-010. 프로토콜 주문
- 실험 프로토콜 텍스트 붙여넣기
- CAS번호/시약명 + 수량 자동 파싱
- 매칭된 제품 일괄 장바구니 추가

#### FR-011. 체크아웃 (2단계)
- **1단계:** 배송 정보 + 청구 정보 + 결제 방법
  - 배송지 선택/추가 (건물, 실험실, 부서 필드)
  - 청구지 분리 관리 (산학협력단 등)
  - PO번호 자동생성 (날짜+사용자명)
  - 이메일 CC 발송 설정
  - 배송 요청사항 자유 텍스트
- **2단계:** 검토 및 제출
  - 주문 요약 확인
  - 견적 요청 / 주문 제출 분리

### 2-4. 주문 관리

#### FR-012. 주문 내역
- 기간 필터 (1주/1개월/3개월/6개월 + 날짜 직접 선택)
- 상태 필터: 전체/결제대기/결제완료/배송준비/배송중/배송완료
- 주문번호/제품명 검색
- XLS/CSV 내보내기
- 주문 상세 모달 (주문 품목, 배송 추적, 타임라인, 배송지, 결제 정보)

#### FR-013. 결제하기
- 탭: 결제대기 / 승인완료 / 거절
- 주문별 승인/거절 버튼 (조직관리자)
- 일괄 승인 기능

#### FR-014. 취소/반품 내역
- 유형 필터: 전체/취소/반품
- 상태 필터: 전체/접수중/처리중/완료
- 취소/반품 신청 폼 (사유 선택, 상세 설명)
- 환불 금액 표시

#### FR-015. 증빙서류
- 기간별 주문 조회 (1주~6개월)
- 체크박스 선택 → 서류 발급
- 견적서 / 거래명세서 / 납품확인서
- 사업자등록증 / 통장사본 다운로드

### 2-5. 내 연구실

#### FR-016. 즐겨찾기
- 시약 즐겨찾기 추가/삭제 (리스트 뷰에서 하트 토글)
- 즐겨찾기 모아보기 그리드
- 바로 재주문 버튼

#### FR-017. 내 시약장
- 보유 시약 등록/관리 (이름, CAS, 수량, 위치, 유효기간)
- 재고 부족 알림 (대시보드 연동)
- 유효기간 만료 알림
- 통계: 총 보유량, 부족 항목, 만료 임박

### 2-6. 커뮤니케이션

#### FR-018. 대화
- 공급사/관리자와 1:1 채팅
- 채팅방 목록 (최근 메시지 미리보기)
- 메시지 입력 + 전송

#### FR-019. 고객센터 홈
- FAQ 검색 + 인기 키워드
- 카테고리별 FAQ 바로가기 (주문/배송/제품/계정/기타)
- TOP 5 FAQ (아코디언)
- 공지사항 리스트
- 빠른 바로가기 (주문내역, 견적내역, 결제하기, 1:1 문의)
- 전화/이메일 상담 안내

#### FR-020. 자주 묻는 질문
- 카테고리 탭 필터 (전체/주문/배송/제품/계정/기타)
- 검색 기능
- 아코디언 펼치기/접기

#### FR-021. 1:1 문의하기
- 문의 작성 폼 (유형, 관련 주문번호, 제목, 내용, 파일 첨부)
- 관련 주문번호: 내 주문 내역에서 선택 가능
- 문의 내역 조회 (상태: 접수/처리중/답변완료)
- 채팅 스타일 답변 표시

### 2-7. 대시보드

#### FR-022. 대시보드 위젯
- **통계 카드:** 장바구니(금액), 주문내역(건수), 배송중(건수), 이번달 주문(금액+전월비교)
- **최근 주문:** 최근 3건 테이블 + "전체보기" 링크
- **빠른 실행:** 프로토콜 주문, 새 견적 요청, 재구매, 카탈로그 검색
- **이번 달 예산:** 진행률 바 + 시약/소모품 분배
- **자주 구매한 제품:** 최근 3개월 기반 TOP 3 + "전체보기" 링크
- **재고 부족 알림:** 내 시약장 연동

### 2-8. 공통 기능

#### FR-023. 최근 본 상품
- 우측 하단 패널 (쿠팡 스타일)
- 리스트 뷰에서만 표시
- 중복 제거, 최대 10개
- 접기/펼치기 + 스크롤

#### FR-024. 상품 비교
- 최대 4개 제품 비교
- 하단 비교 바 → 비교 모달 (테이블 형식)

#### FR-025. 다크 모드
- CSS custom properties 기반 테마 전환
- 토글 버튼 (상단 바)

### 2-9. 외부 API 및 데이터 플랫폼

#### FR-026. 외부 API v1
- 12개 엔드포인트: 제품 검색/상세, 가격 조회, 재고 확인, 추천, 예측, 분석, 지식 그래프 등
- API Key 인증 (Organization 단위 발급)
- 티어별 Rate Limiting: Free(100/일) / Basic(1,000/일) / Pro(10,000/일) / Enterprise(무제한)
- 응답 형식: JSON, 페이지네이션 지원
- 버전 관리 (`/api/v1/...`)

#### FR-027. AI 시약 추천
- 구매 이력(purchase history) + 제품 속성(product attributes) 기반 추천
- 연구 분야별 맞춤 추천 (유기, 무기, 생화학 등)
- 추천 점수(score) + 사유(reason) 제공
- 모델 버전 관리 + 만료일(expires_at) 설정
- Claude API 활용

#### FR-028. AI 소모량 예측
- 소모량 예측 (consumption prediction): 과거 사용량 기반
- 예산 예측 (budget prediction): 월별/분기별 지출 예측
- 재주문 시점 예측 (reorder prediction): 최적 주문 시점 제안
- 신뢰도(confidence) 점수 포함
- Claude API 활용

#### FR-029. AI 분석 리포트
- 구매 패턴 분석 (purchase pattern): 카테고리별, 공급사별, 시간대별
- 트렌드 분석 (trend): 인기 시약, 성장 카테고리, 가격 변동
- 계절성 분석 (seasonality): 연간 주기 패턴
- 기간 설정 (period_start / period_end)
- 조직 단위 리포트 생성

#### FR-030. 화학 지식 그래프
- 시약 간 관계 (대체품, 전구체, 부산물, 용매 호환성)
- 물성 정보 (properties JSON): 끓는점, 녹는점, 밀도, pH 등
- 안전 호환성 매트릭스 (함께 보관 불가 시약 경고)
- CAS 번호 기반 조회
- 출처(source) 명시 + 갱신일(updated_at) 관리

#### FR-031. 데이터 수집 파이프라인
- **DataEvent**: 사용자 행동 이벤트 수집 (페이지 뷰, 클릭, 주문, 검색 등)
  - source_app ENUM: shop / eln / supplier / admin
  - payload JSON: 이벤트별 상세 데이터
- **SearchLog**: 검색 로그 수집 (쿼리, 검색 유형, 결과 수, 클릭 제품)
- **PriceHistory**: 가격 변동 이력 (정가/할인가 스냅샷)

#### FR-032. 개발자 포털
- API 문서 (엔드포인트, 파라미터, 응답 형식, 예제 코드)
- API Key 관리 (발급, 갱신, 비활성화)
- 사용량 대시보드 (일별/월별 호출 수, 엔드포인트별 통계)
- 에러 로그 조회
- 인터랙티브 API 테스터

### 2-10. 공급사 포털

#### FR-033. 공급사 포털
- **대시보드**: 매출 통계, 주문 현황, 견적 현황, 인기 제품 TOP 5
- **견적 관리**: 견적 요청 수신 → 가격/납기 응답 → 주문 전환 추적
- **주문 관리**: 수주 목록, 주문 상세, 배송 상태 업데이트, 출고 처리
- **제품/가격/재고 관리**: 카탈로그 등록/수정, 포장단위별 가격 설정, 실시간 재고 동기화

### 2-11. 관리자 플랫폼

#### FR-034. 관리자 플랫폼
- **운영 대시보드**: 전체 거래량, 활성 사용자, 매출, 주문 상태 분포
- **사용자 관리**: 사용자/기관/역할 CRUD, 권한 관리, 활동 로그
- **제품 관리**: 전체 카탈로그 관리, 카테고리 관리, 공급사 매핑
- **AI 모니터링**: 추천/예측 정확도 대시보드, 모델 버전 관리, AI 사용량 통계
- **API 관리**: API Key 발급/관리, 사용량 모니터링, Rate Limit 설정, 에러 로그

## 3. 비기능 요구사항

### 3-1. 성능
- 페이지 로드: 3초 이내 (LCP)
- 검색 자동완성: 200ms 이내 응답
- 카탈로그 페이지: 무한 스크롤 또는 20개 단위 페이지네이션

### 3-2. 보안

## 6. 제약사항 및 가정

### 제약사항
- 시약은 위험물 분류에 따라 배송 제한이 있음 (항공규제, 택배금지 등)
- 일부 공급사는 API를 제공하지 않아 수동 연동 필요
- 연구기관 예산 집행은 회계연도(1~12월) 기준

### 가정
- 초기 공급사 4~5곳으로 시작, 점진적 확대
- 사용자 대부분은 데스크톱 환경에서 사용
- 기관별 구매 승인 프로세스는 커스터마이징 가능해야 함
