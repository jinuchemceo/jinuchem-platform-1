# IA — JINUCHEM 통합 플랫폼 정보 구조 설계서

> **버전:** 2.0
> **작성일:** 2026-03-19

---

## 0. 전체 플랫폼 사이트맵

```
JINUCHEM 통합 플랫폼 (Turborepo Monorepo)
│
├─ apps/
│   ├─ JINU Shop (shop)          ← B2B 커머스 (모객 창구 / 생태계 허브)
│   │   └─ 연구원·조직관리자 대상
│   │
│   ├─ JINU E-Note (eln)         ← 전자실험노트 (ELN)
│   │   └─ 연구원 대상
│   │
│   ├─ Supplier Portal (supplier) ← 공급사 포털
│   │   └─ 공급사 대상
│   │
│   ├─ Admin Platform (admin)     ← 관리자 플랫폼
│   │   └─ 시스템관리자 대상
│   │
│   └─ External API (api)         ← 외부 API v1
│       └─ 외부 개발자/기관 대상
│
├─ packages/
│   ├─ ui/                        ← 공유 UI 컴포넌트
│   ├─ db/                        ← Prisma 스키마 + 클라이언트
│   ├─ auth/                      ← Supabase Auth (SSO)
│   ├─ ai/                        ← AI 엔진 (Claude API)
│   │   ├─ recommend/             ← 시약 추천
│   │   ├─ predict/               ← 소모량 예측
│   │   ├─ analyze/               ← 분석 리포트
│   │   └─ knowledge/             ← 화학 지식 그래프
│   ├─ config/                    ← 공유 설정
│   └─ utils/                     ← 공유 유틸리티
│
└─ 데이터 플랫폼 (수집 → AI 생성 → API 제공)
    ├─ DataEvent (행동 이벤트)
    ├─ SearchLog (검색 로그)
    └─ PriceHistory (가격 이력)
```

## 2. 사용자 플로우

### 2-1. 시약 구매 플로우

```
[대시보드]
    │
    ├── 빠른 실행 > "카탈로그 검색" ──→ [시약 주문]
    ├── 빠른 실행 > "프로토콜 주문" ──→ [프로토콜 모달]
    ├── 빠른 실행 > "재구매" ─────────→ [자주 구매한 제품]
    │
    ▼
[시약 주문 카탈로그]
    │  검색 / 필터 / 정렬
    ▼
[시약 카드 클릭]
    ▼
[시약 상세 페이지]
    │  사이즈 선택 / 수량 입력
    ├── "장바구니" ──→ [장바구니] ──→ [체크아웃] ──→ [주문 완료]
    ├── "바로 주문" ──→ [체크아웃]
    └── "벌크 견적" ──→ [견적 요청]
```

### 2-2. 빠른 주문 플로우

```
[빠른 주문]
    │
    ├── 제품번호/CAS 입력 → 자동완성 → 테이블 추가
    │                                      │
    ├── CSV 파일 업로드 ──→ 일괄 파싱 ──→ 테이블 추가
    │                                      │
    └── 구조식 검색 ──→ 결과 선택 ──→ 테이블 추가
                                           │
                                           ▼
                                    [장바구니에 추가]
                                           │
                                           ▼
                                      [장바구니]
```

### 2-3. 주문 관리 플로우

```
[주문 내역]
    │
    ├── 주문 상세 보기 → [주문 상세 모달]
    │                     ├─ 주문 품목
    │                     ├─ 배송 추적
    │                     ├─ 배송지 정보
    │                     └─ 결제 정보
    │
    ├── 취소 요청 → [취소/반품 내역] → 신청 폼
    │
    └── 증빙서류 → [증빙서류] → 견적서/명세서/납품확인서 발급
```

### 2-4. 조직 승인 플로우

```
[연구원] 주문 제출
    │
    ▼
[결제하기] — 결제대기 탭
    │
    ▼
[조직관리자] 검토
    ├── 승인 → 결제 진행 → 주문 확정
    └── 거절 → 사유 전달 → 연구원 재수정
```

```
┌────────────────────────────┐

│     · 소모품 주문          │
│  ⚡ 빠른 주문              │
│  📋 주문 관리           ▾  │
│     · 주문 내역            │
│     · 결제하기             │
│     · 취소/반품 내역       │
│  📄 증빙서류              │
│  ─────────────────────    │
│  🔬 내 연구실           ▾  │
│     · 즐겨찾기             │
│     · 내 시약장            │
│  💬 대화                  │
```

## 10. 플랫폼 간 연동 플로우

### 10-1. 실험 기반 시약 주문 사이클

```
[E-Note] 프로토콜 작성
    │ 필요 시약 목록 (ProtocolReagent)
    ▼
[E-Note] CAS 번호 → Shop 제품 자동 매칭
    │ is_matched = true
    ▼
[Shop] 장바구니 일괄 추가
    │
    ▼
[Shop] 주문 제출 → 승인 대기
    │
    ▼
[Supplier Portal] 견적 응답 → 주문 확정
    │
    ▼
[Supplier Portal] 출고 → 배송
    │
    ▼
[Shop] 배송 완료 → LabInventory 자동 입고
    │
    ▼
[E-Note] 실험 진행 → 시약 사용 기록 (UsageRecord)
    │ LabInventory 수량 차감
    ▼
[데이터 플랫폼] DataEvent 수집 → AI 분석
    │
    ▼
[AI] 소모량 예측 → 재주문 알림 → [Shop] 재주문
```

### 10-2. 데이터 순환 플로우

```
[사용자 행동]
    │
    ├── Shop: 검색, 조회, 주문, 장바구니
    ├── E-Note: 실험, 시약 사용
    ├── Supplier Portal: 견적, 주문 처리
    │
    ▼
[데이터 수집] DataEvent + SearchLog + PriceHistory
    │
    ▼
[AI 엔진] Claude API
    ├── Recommend: 시약 추천
    ├── Predict: 소모량/예산/재주문 예측
    ├── Analyze: 구매 패턴/트렌드/계절성 분석
    └── Knowledge: 화학 지식 그래프
    │
    ▼
[2차 데이터] AiRecommendation + AiPrediction + AiAnalytics + ChemKnowledge
    │
    ├── → Shop (추천 표시, 재주문 알림)
    ├── → E-Note (프로토콜 시약 추천)
    ├── → Admin (모니터링 대시보드)
    └── → 외부 API v1 (외부 기관 제공)
```
