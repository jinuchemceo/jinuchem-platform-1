# JINUCHEM 통합 플랫폼 - Claude 작업 지침

> **최종 갱신:** 2026-03-20
> **프로젝트:** JINUCHEM 통합 플랫폼 - 4개 플랫폼 + 데이터 플랫폼 + AI 생태계

---

## 1. 프로젝트 개요

### 1-1. 목적
연구기관(대학·기업 연구소)의 시약 및 실험 소모품 구매를 위한 B2B 커머스 플랫폼에서 출발하여,
**4개 플랫폼 + 데이터 플랫폼 + AI 엔진 + 외부 API**를 아우르는 연구 데이터 생태계로 확장.

JINU Shop은 **모객 창구(Customer Acquisition Gateway)** 이자 **생태계 허브(Ecosystem Hub)** 역할을 수행하며,
기존 시약 유통 플랫폼(Merck/Sigma-Aldrich, Thermo Fisher/Alfa Aesar, 세진씨아이/TCI)을 벤치마킹하여
JINUCHEM 고유의 B2B 워크플로우(역할별 권한, 조직 승인, 공유 장바구니)와 결합.

#### 4개 플랫폼 구성
| 플랫폼 | 역할 | 사용자 |
|--------|------|--------|
| **JINU Shop** | B2B 시약/소모품 커머스 (모객 창구) | 연구원, 조직관리자 |
| **JINU E-Note** | 전자실험노트(ELN) + 시약장 연동 | 연구원 |
| **Supplier Portal** | 공급사 대시보드, 견적/주문/재고 관리 | 공급사 |
| **Admin Platform** | 운영 대시보드, 사용자/제품/AI/API 관리 | 시스템관리자 |

모든 플랫폼은 **Supabase Auth 기반 SSO**로 통합 인증.

### 1-2. 핵심 차별화
- **역할별 권한 분리** — 연구원 / 조직관리자 / 공급자 / 시스템관리자
- **조직 구매 승인 워크플로우** — B2B 연구기관 구매 프로세스 완전 지원
- **공유 장바구니** — 연구팀 협업 구매
- **견적 현황 관리** — 견적대기 → 견적도착 → 주문완료 단계 추적
- **지누 E-Note(전자노트) 연동** — 시약장 ↔ ELN 양방향 연동
- **데이터 플랫폼** — 거래/검색/가격 데이터 수집 → AI 2차 데이터 생성 → 외부 API 제공
- **AI 엔진** — 시약 추천, 소모량 예측, 분석 리포트, 화학 지식 그래프 (Claude API)
- **외부 API v1** — 12개 엔드포인트, API Key 인증, Rate Limiting, 개발자 포털

### 1-3. 기술 스택

#### 프로토타입
- **프론트엔드:** Vanilla HTML/CSS/JS (단일 파일 SPA 구조)
- **스타일:** CSS Custom Properties 기반 테마 시스템 (라이트/다크)
- **아이콘:** Lucide Icons + SVG inline
- **폰트:** Noto Sans KR (Google Fonts)
- **서버:** 정적 파일 서빙 (live-server)

#### 프로덕션
- **모노레포:** Turborepo (apps/ + packages/)
- **프론트엔드:** Next.js 15 (App Router) + React 19 + TypeScript
- **상태 관리:** Zustand
- **스타일:** Tailwind CSS 4
- **아이콘:** lucide-react
- **유효성 검증:** Zod
- **DB:** PostgreSQL (Supabase)
- **ORM:** Prisma
- **인증:** Supabase Auth (SSO) + API Key 시스템 (외부 API)
- **AI:** Claude API (추천, 예측, 분석, 지식 그래프)
- **배포:** Vercel (프론트엔드) + Supabase (백엔드/DB)

## 5. 디자인 원칙

### 5-1. 테마
- **라이트 모드 기본** (사이드바도 라이트)
- 다크 모드 토글 지원 (CSS custom properties)
- Primary 색상: `#3b82f6` (파란색)
- Accent: `#10b981` (초록), `#f59e0b` (노란), `#ef4444` (빨간)

### 5-2. 레이아웃
- 사이드바: 250px 고정폭, 햄버거 버튼으로 접기 가능
- 메인 콘텐츠: `margin-left: var(--sidebar-width)`
- 상단 바: JINU shop 브랜드 + 다크모드 + 장바구니(배지) + 알림 + 유저 메뉴
- 푸터: YES24 스타일 — 회사정보 + 고객센터 + E-Note 링크

### 5-3. UI 규칙
- **이모지 사용 금지** — 모든 아이콘은 SVG 또는 Lucide Icons 사용
- **버튼 높이 통일: 38px** — 검색바, 드롭다운, 액션 버튼 모두 동일
- **카드 기반 레이아웃** — 그리드(기본) / 리스트 뷰 전환 가능
- **페이지네이션** — 원형 번호 버튼 (1, 2, 3... > >>)
- **토스트 알림** — 장바구니 담기 등 피드백

### 5-4. 시약 카드 표시 규칙
- 상단: **공급사명** (카드 상단 좌측 배지)
- 중앙: 구조식 이미지 (크게)
- 하단: 제품명, CAS No., 분자식, 가격
- 하트(즐겨찾기): **리스트 뷰에서만 표시** (그리드 카드에서는 숨김)
- 물질명 옆 하트: 없음

### 5-5. 소모품 카드 표시 규칙
- 이미지 없음 (텍스트 기반 카드)
- 카테고리: 단위 없이 **이름과 품목수만** 표시
- 정기배송 옵션: 기간 설정 포함
- 구조는 시약 주문과 동일한 레이아웃

## 6. 핵심 기능 상세

### 6-1. 검색
- 통합 검색바: 시약명, CAS번호, 분자식, 카탈로그번호
- CAS번호 패턴 자동 감지 (`XX-XX-X` 형태)
- 구조식 검색 모달 (Draw / SMILES / Formula 탭)
- 공급사 검색 필터 (공급사 목록 내 검색)

### 6-2. 주문 프로세스
```
카탈로그 검색 → 제품 상세 → 장바구니 → 체크아웃(2단계) → 주문완료
                                ↑
빠른 주문 (카탈로그번호/CSV) ──┘
프로토콜 주문 (CAS/수량 일괄) ──┘
```

### 6-3. 문서 관리
| 문서 | 시약 | 소모품 |
|------|:----:|:-----:|
| SDS (한글/영문) | O | O |
| COA | O | - |
| COO | O | - |
| Spec Sheet | O | O |
| KC 인증서 | - | O |

### 6-4. 가격 표시
- 포장단위별 가격 테이블 (TCI 방식)
- 정가/할인가 이중 표시 (Alfa Aesar 방식)
- VAT 자동 계산 (소계 → 할인 → VAT → 총액)
- 납품예정일 정확한 날짜 표시

### 6-5. 배송
- 당일출고 배지 + 마감시간 명시 ("오후 3시 이전")
- 납품예정일 날짜 표시
- 익스프레스 딜리버리 옵션 (추후)

## 8. 작업 시 주의사항

### DO (반드시)
- 버튼 높이 38px 통일
- 이모지 대신 SVG 아이콘 사용
- 카드 공급사명은 카드 상단에 텍스트로 표시

- 카드 그리드에서 하트(즐겨찾기) 표시 금지

- 공급사명 뒤에 특수문자 붙이지 않기
- 빈 페이지 만들지 않기 (모든 페이지에 콘텐츠 필수)

## 9. 플랫폼 생태계 연동

### 9-1. JINU E-Note 연동 (양방향)
- 내 시약장 ↔ 전자노트(ELN) 양방향 동기화
- 실험 프로토콜에서 시약 자동 주문 (ProtocolReagent → Cart)
- 시약 사용 이력 자동 기록 (UsageRecord → LabInventory 차감)
- 실험 진행 중 시약 부족 시 자동 재주문 알림
- 프로토콜 기반 소모량 예측 (AI Predict 엔진 연동)

### 9-2. Supplier Portal 연동
- 공급사 대시보드: 매출, 주문, 견적 현황 실시간 조회
- 견적 요청 → 응답 → 주문 확정 워크플로우 자동화
- 제품/가격/재고 관리 (PriceHistory 자동 기록)
- 공급사 API 연동 (재고·가격 실시간 동기화)

### 9-3. Admin Platform
- 운영 대시보드: 전체 거래량, 사용자, 매출 통계
- 사용자 관리: 역할/권한/기관 관리
- 제품 관리: 카탈로그, 가격, 재고 통합 관리
- AI 모니터링: 추천/예측 정확도, 모델 버전 관리
- API 관리: API Key 발급, 사용량 모니터링, Rate Limit 설정

### 9-4. 외부 API v1
- 12개 엔드포인트 (제품, 가격, 재고, 추천, 예측, 분석)
- API Key 인증 + 티어별 Rate Limiting (Free / Basic / Pro / Enterprise)
- 개발자 포털: API 문서, Key 관리, 사용량 대시보드
- 데이터 수집 파이프라인: DataEvent, SearchLog, PriceHistory

### 9-5. AI 엔진 (Claude API)
- **Recommend** — 구매 이력 + 제품 속성 기반 시약 추천
- **Predict** — 소모량/예산/재주문 시점 예측
- **Analyze** — 구매 패턴, 트렌드, 계절성 분석 리포트
- **Knowledge Graph** — 시약 관계, 호환성, 대체품 화학 지식 그래프

### 9-6. 외부 시스템 연동
- PG사 결제 연동
- 배송 추적 API
- 전자세금계산서 자동 발행
